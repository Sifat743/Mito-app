Deploy on GitHub Pages:
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub: Settings -> Pages.
4. Set Source to 'Deploy from a branch'.
5. Choose branch 'main' and folder '/root', then save.
6. Wait for the Pages URL to be published.
7. Open the Pages URL on your phone in Chrome.
8. Use the browser menu or in-app install flow to add Mito to your home screen.

Important:
- Install prompts work on HTTPS or localhost, not file://.
- Existing local data only carries over on the same browser/device/origin. A new hosted URL starts with fresh storage unless you import backup data.
